Topic 2: Setting Up the Environment

2.1 Why Set Up a Programming Environment?

Before writing any code, you need to set up an environment where you can write, test, and run your programs efficiently. A good setup helps you be more productive and avoid errors.

2.2 Choosing a Programming Language

For this course, we will use Python because it is beginner-friendly and widely used in many fields like web development, data science, and automation.

2.3 Installing Python

  1. Go to the official Python website: https://www.python.org
  2. Download the latest version of Python for your operating system (Windows, Mac, or Linux).
  3. Run the installer and ensure you check the box that says "Add Python to PATH".
  4. Verify installation by opening your terminal or command prompt and typing:
    python --version

2.4 Setting Up a Code Editor

A code editor is where you write your code. Popular editors include:

  • VS Code: Lightweight, customizable, and free. Download from https://code.visualstudio.com/
  • PyCharm: Specifically designed for Python development.
  • Thonny: Simple and ideal for beginners in Python.

2.5 Running Your First Python Program

After installing Python and a code editor:

  1. Open your editor and create a new file named hello.py.
  2. Type the following code:
print("Hello, Smiley Academy!")
  1. Save the file.
  2. Open terminal/command prompt, navigate to the file location, and run:
    python hello.py
  3. You should see:
    Hello, Smiley Academy!

2.6 Summary

  • Choose a programming language (we're using Python).
  • Install Python from the official site.
  • Set up a code editor like VS Code or Thonny.
  • Write and run your first Python program!
