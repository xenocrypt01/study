Topic 3: Variables and Data Types

3.1 What are Variables?

Variables are containers used to store data that can be referenced and manipulated in a program. Think of them as labeled boxes where you store values you want to use later.

Example in Python:
name = "John"
age = 25
is_student = True

3.2 Why Use Variables?

  • To store user input or data fetched from other sources.
  • To perform calculations and keep results.
  • To make code dynamic and reusable.

3.3 Data Types

Data types define the kind of data a variable can hold. Common data types include:

  • String: Text data. Example: "Hello"
  • Integer: Whole numbers. Example: 5
  • Float: Decimal numbers. Example: 3.14
  • Boolean: True or False. Example: True
  • List: A collection of items. Example: [1, 2, 3]

3.4 Dynamic Typing

In languages like Python, you don’t need to declare data types explicitly; the interpreter figures it out for you.

x = 10 (x is an integer)
x = "Hello" (now x is a string)

3.5 Best Practices

  • Use meaningful variable names: user_name, total_price.
  • Follow consistent naming conventions like snake_case.
  • Keep track of the data type you're working with to avoid errors.

3.6 Summary

Variables and data types are fundamental in any programming language. Understanding them is the first step to managing data effectively in your programs.
