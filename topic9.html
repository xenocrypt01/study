Topic 9: Lists and Collections

9.1 What are Lists?

A list is a collection of items that are ordered and changeable. Lists can store multiple data types like numbers, strings, or even other lists.

9.2 Creating a List

fruits = ["apple", "banana", "cherry"]

9.3 Accessing List Items

print(fruits[0]) # Outputs: apple

9.4 Changing List Items

fruits[1] = "blueberry"
print(fruits)

9.5 List Methods

  • append() - Adds an item to the end
  • insert() - Inserts an item at a specified index
  • remove() - Removes the first occurrence of an item
  • pop() - Removes an item by index
  • sort() - Sorts the list
fruits.append("orange")
print(fruits)

9.6 Looping Through a List

for fruit in fruits:
  print(fruit)

9.7 Other Collections

  • Tuples: Immutable lists my_tuple = (1, 2, 3)
  • Sets: Unordered collections without duplicates my_set = {1, 2, 3}
  • Dictionaries: Key-value pairs my_dict = {"name": "Alice", "age": 25}

9.8 Example of a Dictionary

person = {"name": "John", "age": 30}
print(person["name"])

9.9 Summary

Lists and collections help store and manage multiple data items efficiently. Understanding them is crucial for data organization in Python.
