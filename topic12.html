Topic 12: Simple Project - Calculator

12.1 Overview

Let's build a simple calculator in Python that can add, subtract, multiply, and divide numbers.

12.2 Project Requirements

  • Accept user input for two numbers.
  • Provide options to add, subtract, multiply, or divide.
  • Display the result.
  • Handle division by zero gracefully.

12.3 Project Code Example

def add(x, y):
  return x + y

def subtract(x, y):
  return x - y

def multiply(x, y):
  return x * y

def divide(x, y):
  if y == 0:
    return "Cannot divide by zero!"
  return x / y

print("Select operation:")
print("1. Add")
print("2. Subtract")
print("3. Multiply")
print("4. Divide")

choice = input("Enter choice (1/2/3/4): ")
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

if choice == '1':
  print("Result:", add(num1, num2))
elif choice == '2':
  print("Result:", subtract(num1, num2))
elif choice == '3':
  print("Result:", multiply(num1, num2))
elif choice == '4':
  print("Result:", divide(num1, num2))
else:
  print("Invalid input!")

12.4 What You Learned

  • Defining and using functions.
  • Handling user input and choices.
  • Implementing error checks (like divide by zero).

12.5 Try It Yourself

Expand this project by:

  • Adding more operations (e.g., power, modulus).
  • Creating a loop to allow multiple calculations.
  • Building a simple GUI with a library like Tkinter.
