Topic 6: Control Flow (If/Else Statements)

6.1 What is Control Flow?

Control flow allows a program to make decisions and execute certain parts of code based on conditions. This helps create dynamic and responsive programs.

6.2 If Statement

The if statement runs a block of code if a condition is true.

age = 18
if age >= 18:
  print("You are an adult.")

6.3 If-Else Statement

Use else to run an alternative block when the condition is false.

age = 16
if age >= 18:
  print("Adult")
else:
  print("Minor")

6.4 Elif Statement

elif means "else if". It allows multiple conditions to be checked sequentially.

marks = 75
if marks >= 90:
  print("Grade A")
elif marks >= 70:
  print("Grade B")
else:
  print("Grade C")

6.5 Nested If Statements

Conditions can be nested within each other for more complex logic.

num = 10
if num > 0:
  if num % 2 == 0:
    print("Positive Even Number")

6.6 Logical Operators with Conditions

Use and, or, and not to combine conditions.

age = 20
if age >= 18 and age <= 60:
  print("Working Age Group")

6.7 Summary

  • if - Executes when a condition is true.
  • else - Executes when the condition is false.
  • elif - Checks another condition if the previous ones are false.
  • Nested and combined conditions allow for complex decision-making.
