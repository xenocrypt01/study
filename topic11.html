Topic 11: Basic Object-Oriented Programming (OOP)

11.1 What is Object-Oriented Programming?

OOP is a programming paradigm that uses objects and classes to structure code. It helps in organizing complex programs by modeling real-world entities.

11.2 Classes and Objects

  • Class: A blueprint for creating objects.
  • Object: An instance of a class with data and behavior.
class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

person1 = Person("Alice", 30)
print(person1.name)

11.3 The __init__ Method

This special method is called when an object is created. It initializes the object's attributes.

11.4 Methods in Classes

Functions defined inside a class are called methods.

class Dog:
  def __init__(self, name):
    self.name = name

  def bark(self):
    print(f"{self.name} says woof!")

dog1 = Dog("Buddy")
dog1.bark()

11.5 Inheritance

Inheritance allows a class to inherit properties and methods from another class.

class Animal:
  def speak(self):
    print("Animal speaks")

class Cat(Animal):
  def speak(self):
    print("Meow!")

cat1 = Cat()
cat1.speak()

11.6 Encapsulation

Encapsulation means restricting access to methods and attributes to protect data integrity.

class BankAccount:
  def __init__(self, balance):
    self.__balance = balance # private attribute

  def get_balance(self):
    return self.__balance

account = BankAccount(1000)
print(account.get_balance())

11.7 Summary

  • OOP helps in structuring code via classes and objects.
  • Key concepts: Classes, Objects, Methods, Inheritance, Encapsulation.
  • It makes code reusable, organized, and scalable.
