Topic 7: Loops (For/While)

7.1 What are Loops?

Loops allow you to execute a block of code multiple times. They are essential for tasks that require repetition, such as processing items in a list or counting numbers.

7.2 For Loop

A for loop iterates over a sequence like a list, tuple, or range.

for i in range(5):
  print(i)

This will print numbers from 0 to 4.

7.3 While Loop

A while loop continues to execute as long as a condition is true.

count = 0
while count < 5:
  print(count)
  count += 1

7.4 Nested Loops

Loops can be placed inside other loops. This is useful for working with multi-dimensional data like matrices.

for i in range(3):
  for j in range(2):
    print(f"i={i}, j={j}")

7.5 Break and Continue

  • break: Exits the loop immediately.
  • continue: Skips the current iteration and continues to the next.
Break Example:
for i in range(5):
  if i == 3:
    break
  print(i)
Continue Example:
for i in range(5):
  if i == 3:
    continue
  print(i)

7.6 Loop with Else

A loop can have an else block that runs after the loop finishes naturally (without break).

for i in range(3):
  print(i)
else:
  print("Loop finished!")

7.7 Summary

  • Use for when iterating over a sequence.
  • Use while when the number of iterations is unknown.
  • Control loop behavior using break and continue.
  • Loops can have else clauses for completion logic.
