Topic 4: Operators

4.1 Introduction to Operators

Operators are symbols or keywords that perform operations on variables and values. They are essential in programming to compute data, compare results, or control logic flow.

4.2 Types of Operators

4.2.1 Arithmetic Operators

These operators perform basic mathematical operations:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder)
  • ** : Exponentiation
  • // : Floor division
Example:
a = 10 + 5
b = 10 % 3

4.2.2 Comparison Operators

These are used to compare two values and return a boolean result:

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to
Example:
5 == 5 # True
5 < 3 # False

4.2.3 Logical Operators

Used to combine conditional statements:

  • and : Returns True if both statements are true
  • or : Returns True if at least one statement is true
  • not : Reverses the result
Example:
x = 5
y = 10
x > 0 and y > 0 # True

4.2.4 Assignment Operators

These are used to assign values to variables:

  • = : Assign
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
Example:
x = 5
x += 3 # x becomes 8

4.2.5 Bitwise Operators

These operators work on bits and perform bit-by-bit operations:

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Zero fill left shift
  • >> : Signed right shift

4.3 Importance of Operators

  • Facilitate calculations and data manipulation.
  • Help in decision making with comparisons and logic.
  • Enable concise code through compound assignments.

4.4 Summary

Operators are fundamental tools in programming used to process data, make decisions, and control program flow. Mastery of operators allows you to write effective and efficient code.
