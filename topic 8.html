Topic 8: Functions

8.1 What is a Function?

A function is a reusable block of code that performs a specific task. Functions help organize code, make it more readable, and avoid repetition.

8.2 Defining a Function

Functions are defined using the def keyword.

def greet():
  print("Hello, Welcome to Smiley Academy!")

8.3 Calling a Function

After defining a function, you can call it by its name followed by parentheses:

greet()

8.4 Functions with Parameters

Functions can accept parameters to work with external data.

def greet_user(name):
  print(f"Hello, {name}!")

greet_user("Alice")

8.5 Functions with Return Values

Functions can return values using the return statement.

def add(a, b):
  return a + b

result = add(5, 3)
print(result)

8.6 Default Parameter Values

You can set default values for parameters:

def greet(name="Student"):
  print(f"Hello, {name}!")

greet()
greet("Bob")

8.7 Keyword Arguments

Arguments can be passed using parameter names, allowing flexibility in order:

def profile(name, age):
  print(f"Name: {name}, Age: {age}")

profile(age=25, name="Alice")

8.8 Why Use Functions?

  • Reduces code duplication
  • Makes code modular and organized
  • Improves readability and maintenance
  • Enables reusability across projects

8.9 Summary

Functions are essential for writing clean, organized, and efficient code. They encapsulate tasks and allow for code reusability, making programs easier to manage.
