Topic 5: Input and Output

5.1 Introduction to Input and Output

Input and Output (I/O) operations allow interaction between the user and the program. Input captures user data, while output displays information to the user.

5.2 The input() Function

The input() function is used to take data from the user as a string.

Example:
name = input("Enter your name: ")
print("Hello", name)

5.3 Converting Input to Other Data Types

By default, input returns a string. To work with numbers, you must convert them:

age = int(input("Enter your age: "))
height = float(input("Enter your height: "))

5.4 The print() Function

The print() function outputs data to the screen.

print("This is an output message.")
print("You are", age, "years old.")

5.5 Using print() with Formatting

Use formatted strings for cleaner output:

name = "Alice"
print(f"Welcome, {name}!")

5.6 End and Separator in print()

Customize the print behavior with end and sep:

print("Hello", end=" ")
print("World!")
print("A", "B", "C", sep="-")

5.7 Multi-Line Input and Output

You can combine inputs and outputs to create interactive experiences.

name = input("Enter your name: ")
age = input("Enter your age: ")
print(f"{name} is {age} years old.")

5.8 Summary

Input and output are fundamental for interaction in programs. With input() and print(), you can build engaging and user-driven applications.
